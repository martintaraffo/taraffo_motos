# Taraffo Motos

## Visión del proyecto

Taraffo Motos nace con un objetivo claro: construir la principal plataforma de conocimiento sobre mecánica, mantenimiento, viajes y cultura motociclista basada en la experiencia real de Adrián Taraffo.

No buscamos crear solamente la página web de un taller mecánico.

Queremos construir un lugar donde cualquier motociclista pueda aprender, encontrar respuestas confiables y descubrir la experiencia acumulada durante más de cincuenta años de vida entre motos.

El taller es el punto de partida.

El conocimiento es el verdadero producto.

---

## ¿Quién es Adrián Taraffo?

Adrián es ingeniero, mecánico y motociclista desde la década de 1970.

Durante más de cinco décadas trabajó con motos de diferentes marcas y cilindradas, realizó viajes por rutas argentinas y acumuló una enorme experiencia práctica que hoy constituye el mayor valor de este proyecto.

La misión de esta plataforma es preservar, organizar y compartir ese conocimiento para las generaciones actuales y futuras de motociclistas.

---

## Objetivo de la primera versión

La primera versión del sitio tiene como propósito ofrecer una presencia profesional que complemente el contenido publicado en Instagram.

El visitante debe poder:

* conocer quién es Adrián Taraffo;
* conocer el taller;
* ver los servicios ofrecidos;
* acceder rápidamente al contacto;
* explorar fotografías del taller y de distintos trabajos realizados;
* comenzar a descubrir contenido técnico de calidad.

La experiencia debe estar especialmente optimizada para dispositivos móviles, ya que la mayor parte del tráfico provendrá desde Instagram.

---

## Visión a largo plazo

Esta plataforma crecerá progresivamente hasta convertirse en un ecosistema de contenido para motociclistas.

Entre las futuras funcionalidades se contemplan:

* biblioteca de artículos técnicos;
* restauraciones documentadas paso a paso;
* relatos de viajes en moto;
* preguntas frecuentes;
* recursos descargables;
* guías en PDF;
* cursos en video;
* sistema de búsqueda;
* cuentas de usuario;
* historial de mantenimiento de cada motocicleta;
* recordatorios de servicios;
* herramientas digitales para clientes del taller.

Cada nueva función deberá aportar valor real al motociclista y fortalecer la confianza en la experiencia de Adrián Taraffo.

---

## Filosofía del proyecto

Este sitio no pretende competir por cantidad de contenido.

Pretende diferenciarse por la calidad, la claridad y la experiencia real detrás de cada publicación.

Cada artículo, video o recurso deberá responder una necesidad concreta de los motociclistas y estar basado en situaciones reales del taller o de la experiencia personal de Adrián.

El objetivo no es generar visitas pasajeras.

El objetivo es construir una comunidad de personas que quieran comprender mejor sus motos, cuidarlas correctamente y disfrutar de la cultura motociclista.

## Tecnologías

En esta primera etapa el proyecto se desarrollará utilizando:

* HTML
* CSS
* JavaScript
* Git
* GitHub

A medida que el proyecto evolucione podrán incorporarse nuevas tecnologías cuando exista una necesidad real que las justifique.

---

## Forma de trabajo

Este proyecto se desarrolla siguiendo una metodología incremental.

Cada sesión de trabajo debe cumplir el siguiente ciclo:

1. Definir un objetivo concreto.
2. Diseñar la solución antes de escribir código.
3. Implementar únicamente lo necesario.
4. Verificar el funcionamiento.
5. Registrar los cambios mediante un commit con un mensaje descriptivo.
6. Documentar el avance.

La prioridad siempre será comprender cada decisión antes de implementarla.

---

## Principio rector

Antes de incorporar cualquier funcionalidad, deberá responderse una única pregunta:

**¿Esta mejora aporta valor real al motociclista y aumenta la confianza en la experiencia de Adrián Taraffo?**

Si la respuesta es sí, probablemente forme parte del proyecto.

Si la respuesta es no, deberá replantearse o dejarse para una etapa futura.
